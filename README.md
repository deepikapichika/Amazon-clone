# Amazon Clone Project

## Overview
This is a **static Amazon clone homepage** built using **HTML and CSS**.  
It replicates Amazon’s homepage layout with product cards, navigation bar, and basic styling.

## Features
- Responsive homepage design
- Product listing grid
- Navigation bar with logo and search bar
- Clean, organized HTML and CSS structure

## Tech Stack
- **Languages:** HTML, CSS
- **Tools:** VS Code, Git, GitHub
